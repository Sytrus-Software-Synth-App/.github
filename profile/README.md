## 🚀 At a Glance
Sytrus for Windows is Image-Line's multi-engine synthesizer, a mainstay of the FL Studio ecosystem for well over a decade. Running as the Sytrus VST inside FL Studio and other compatible hosts, it lets you combine six operators with ring modulation and a subtractive stage to design tones ranging from glassy bells to fat leads. Producers reach for the fl studio sytrus workflow whenever they want a single instrument that can cover FM, RM, and subtractive territory at once.

## ✨ What You Get
| Capability | What It Means for You |
|------------|-----------------------|
| **FM + ring modulation** | Six operators and a modulation matrix produce rich, harmonically complex tones. |
| **Subtractive engine** | Filters and oscillator shaping give the Sytrus synth analog-style warmth alongside its FM edge. |
| **Per-operator control** | Dedicated envelopes, waveforms, and unison shape each operator independently. |
| **Deep modulation** | LFOs, envelopes, and mapping curves route to nearly any parameter for evolving patches. |
| **Built-in effects** | Onboard reverb, delay, chorus, and EQ finish sounds without extra plugins. |

## 🎯 Quick Start
1. Load Sytrus onto a channel in FL Studio or a compatible host.
2. Explore the factory presets to hear how far the engine reaches.
3. Adjust the operator matrix to change how the operators modulate each other.
4. Bring in LFOs and envelopes for movement, then apply the built-in effects.
5. Save your work as a preset and use it in your tracks.

## ❓ FAQ
**Is Sytrus free?**
Sytrus is included with certain FL Studio editions and available as a paid add-on for others. The FL Studio trial lets you try it before you buy.

**Which versions of Windows are supported?**
Sytrus for Windows runs on current 64-bit versions of Windows, including Windows 10 and Windows 11, when hosted in a compatible DAW.

**Sytrus vs Serum — what's the difference?**
Sytrus is an FM/RM and subtractive synth built around operators, while Serum is a wavetable synth. Both are highly capable, so the choice depends on whether FM or wavetable sound design suits your music.
